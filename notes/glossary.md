# IAM Glossary

## Authentication
Process of verifying the identity of a user or system.

## Authorization
Process of determining what an authenticated user is allowed to access.

## OAuth 2.0
Authorization framework that allows applications to obtain limited access to resources on behalf of a user.

## OpenID Connect (OIDC)
Identity layer built on top of OAuth 2.0 that provides user authentication information.

## JWT
JSON Web Token used to securely transmit claims between parties.

## Claim
A piece of information about a user contained within a token.

## Scope
Permissions requested by a client application.

## Identity Provider (IdP)
System responsible for authenticating users.

## Service Provider (SP)
Application that relies on an IdP for authentication.

## Single Sign-On (SSO)
Allows a user to log in once and access multiple applications.
